Koers voorspellen

Voorspellen van de aandelen koers

Klik hier op het project te bekijken op Github

Het voorspellen van de aandeelprijs is natuurlijk erg interessant. Een aandeelprijs is afhankelijk van vele factoren, waarbij de psychologie van de massa een hoofdrol speelt. Sinds de opkomst van kunstmatige intelligentie, wordt er geprobeerd om mathematisch de koers te voorspellen doormiddel van een statistisch model.

In het huidige project wordt een poging gedaan, aan de hand van een aantal zogenoemde technische indicatoren de prijs te voorspellen.

Er was naast de notebook ook een webapp ontworpen in Python en gepubliseerd op Heroku. Helaas is deze app niet meer functioneel. De broncode is uiteraard nog wel terug te vinden in de Github repo.
