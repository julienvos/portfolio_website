Regen in Australië

Weersvoorspelling Steden

Klik hier op het project te bekijken op Google Collab

Een veelgebruikte techniek om het weer te voorspelling is doormiddel van machine Learning. Op basis van data over luchtvochtigheid, bewolking, zonneuren, windsnelheid etc. kan een statistisch model worden gecreërd om een gegronde uitspraak te kunnen doen over het weer.

In dit project wordt op basis van meer dan 10 jaar aan data uit verschillende Australische steden een statistisch model gemaakt. In de eerste fase van het project wordt natuurlijk eerst de data bekeken doormiddel van een zogenoemde Exploratory Data Analysis. Vervolgens wordt de data bewerkt met als doel om patronen sneller te herkennen tijdens het 'leren', oftwel het maken van een machine learning model. Dit kan bijvoorbeeld door de gemiddelde neerslag per week te berekenen vanuit de dagelijkse neerslag.

In het project wordt gebruik gemaakt van pakketten/tools binnen de programmeertaal python. De databron is kaggle.
