Android/IOS app met Flutter

Klik hier op het project te bekijken op Github

Mobiele apps zijn tegenwoordig niet meer weg te denken vandaag de dag. Naast het feit dat het aantal apps met de dag groeit, groeien ook de mogelijkheden om apps te bouwen. Een vrij nieuwe methode van apps bouwen op zowel Android als IOS is doormiddel van Dart en het bijbehorende framework Flutter. Dit systeem is ontwikkeld door Google en heeft de ambitie om met één keer de code voor een app te schrijven, waarnaar dit gepubliseerd kan worden over verscheidene systemen (o.a. web/IOS/Android/Windows).

In het huidige project wordt doormiddel van Flutter een nieuws app gebouwd. Het gaat hierbij om een app voor zowel Android als IOS. De gebruiker wordt verwelkomd door een introductie scherm, kan vervolgens inloggen en krijgt daarna een intuitive interface om nieuws content te bekijken.

Flutter vind ik persoonlijk een toegankelijk framework, waar je al snel zelf projecten kan in starten. Qua programmeren is het een combinatie van Java en Javascript. Een app is hierbij opgebouwd uit zogenoemde Widgets (in React zijn dat components).
