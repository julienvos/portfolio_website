Portfolio website

HTML/CSS/Javascript

Klik hier op het project te bekijken op Github

Voor het maken van een portfolio website heb ik eerst gedacht om dit via Wordpress te doen. Door eerste de nodige research te doen, werd door vele bronnen aanbevolen om een portfolio pagina in Wordpress te doen. Wordpress wordt gekenmerkt doordat je relatief snel een website in elkaar flanst. Ik heb geprobeerd om in Wordpress te werken, maar kwam er al snel achter dat je een beetje het gevoel van controle en creativiteit kwijt bent.

Daarom heb ik vervolgens besloten om mijn website in HTML/CSS en een klein beetje Javascript te doen. Voordat ik mijn project begon had ik amperervaring in HTML en CSS en zag het daarom als een leer traject.

Doormiddel van instructie filmpjes op Youtube heb ik uiteindelijk mijn project kunnen opzetten. Ik heb veel geleerd hierover en wordt steeds handiger doordat ik tegen steeds minder dingen aanloop. Dit was ook mijn doel met het project en ik ben blij met het resulaat.
